# Curated seed list of popular crates.io crates for typosquat detection.
#
# Source: compiled from crates.io's most-downloaded crate rankings
# (https://crates.io/crates?sort=downloads) as of the offline-corpus
# bootstrap. crates.io exposes a live top-N download API (used by
# fetchCargo when the network is reachable), but air-gapped / install-
# hot-path deployments cannot reach it, so — like the Go, CocoaPods and
# pub seeds — we ship a curated file and fall back to it offline. This
# keeps the detector bootstrap deterministic while still preferring the
# live feed when one is available.
#
# Format:
#   - Blank lines and lines starting with '#' are ignored.
#   - One crate name per line, no version suffix.
#   - crates.io names are flat (no scopes), case-insensitive; case is
#     preserved here but the detector lowercases at lookup time.
#
# Ultra-short / ambiguous names are deliberately omitted to avoid
# false-positive edit-distance matches against unrelated short names.
