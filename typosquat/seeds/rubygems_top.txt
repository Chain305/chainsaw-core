# Curated seed list of popular RubyGems gems for typosquat detection.
#
# Source: compiled from rubygems.org most-downloaded gem rankings
# (https://rubygems.org/stats) as of the offline-corpus bootstrap.
# rubygems.org exposes a live search/download API (used by fetchRubyGems
# when the network is reachable), but air-gapped / install-hot-path
# deployments cannot reach it, so — like the Go, CocoaPods and pub seeds —
# we ship a curated file and fall back to it offline. This keeps the
# detector bootstrap deterministic while still preferring the live feed
# when one is available.
#
# Format:
#   - Blank lines and lines starting with '#' are ignored.
#   - One gem name per line, no version suffix.
#   - RubyGems names are flat (no scopes), case-insensitive; case is
#     preserved here but the detector lowercases at lookup time.
#
# Ultra-short / ambiguous names are deliberately omitted to avoid
# false-positive edit-distance matches against unrelated short names.
